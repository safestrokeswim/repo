Setup Instructions

  1. Database Setup: Run the SQL scripts in Supabase:
    • First run database-schema-complete.sql
    • Then run generate-time-slots.sql (optional, or use the button above)
  2. Stripe Webhook: Configure webhook endpoint in Stripe Dashboard:
    • Endpoint URL: https://your-site.netlify.app/.netlify/functions/stripe-webhook
    • Events to listen: payment_intent.succeeded, payment_intent.payment_failed
    • Add webhook secret to environment variables as STRIPE_WEBHOOK_SECRET
  3. Environment Variables: Ensure all are set in Netlify:
    • STRIPE_SECRET_KEY
    • STRIPE_PUBLISHABLE_KEY
    • STRIPE_WEBHOOK_SECRET
    • SUPABASE_URL
    • SUPABASE_SERVICE_KEY
  4. Test the System: Use the "Create Test Package" feature above to generate a test package code
